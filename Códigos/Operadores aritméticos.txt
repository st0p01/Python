                         Operadores aritiméticos
_____________________________________________________________________________
+ = Adição          ** = Potência 
- = Subtração       // = Divisão inteira
* = Multiplicação   % = Resto da Divisão
/ = divisão 

5 + 2 = 7        5 ** 2 = 25
5 - 2 = 3        5 // 2 = 2
5 * 2 = 10       5 % 2 = 1
5 / 2 = 2.5

_____________________________________________________________________________
                          Ordem de Precedência
1 = ()
2 = **
3 = * / // %
4 = + -
_____________________________________________________________________________
